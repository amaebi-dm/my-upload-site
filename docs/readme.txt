・PLYアップロード画面を作る。
　※PLY表示画面とは別のHTML。
・フォームでファイルを受け取る。
・Xserverへファイルを保存。
　（おそらくCGIが必要。）
・PLYからSPZへコンバートしXserverへ保存。
　Xserver内のファイルリストにコンバートしたファイルを追加。
　（おそらくCGIが必要。）
・SPZとして使用できるファイルのURLをスプレッドシートへ書き込む。
　（スプレッドシートで更新ボタンを押してファイルリストを取り込む。）
・スプレッドシート上で、カメラ移動ポイント、カメラ移動ポイント名、カメラ移動範囲等を設定。
・スプレッドシートからjsonを吐き出してXserverへ保存。
・表示HTMLのURLパラメータでjsonを指定するとSPZが表示される。
